# Azure AD Hybrid Identity Migration Lab

## Project Overview
Migrated on-premises Windows Server Active Directory 
to Azure AD (Microsoft Entra ID) with hybrid identity.

## Technologies Used
- Microsoft Azure (IaaS)
- Windows Server 2025
- Active Directory Domain Services (AD DS)
- Microsoft Entra Connect Sync
- Password Hash Synchronisation (PHS)
- Self-Service Password Reset (SSPR)

## What I Built
1. Deployed Windows Server 2025 VM on Azure
2. Configured Active Directory and Domain Controller
3. Created domain users (corp.local forest)
4. Installed Microsoft Entra Connect Sync
5. Enabled Password Hash Synchronisation
6. Configured Self-Service Password Reset
7. Verified user sync to Azure AD

## Challenges Overcome
- Azure subscription quota limits
- VM availability zone restrictions
- OAuth browser security configuration
- MFA setup for admin account
- Active Directory replication timing
